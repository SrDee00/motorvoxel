# Change: Add Post-Processing Effects

## Why
Post-processing effects are essential for enhancing visual quality and creating immersive experiences in the voxel world. They enable advanced visual effects that significantly improve the game's aesthetic appeal and atmosphere.

## What Changes
- Implement post-processing pipeline
- Add screen-space effects (SSAO, bloom, etc.)
- Implement color grading
- Add depth of field effects
- Implement motion blur
- Add post-processing configuration system

## Impact
- Affected specs: rendering
- Affected code: New post-processing system in src/rendering/
- This will significantly enhance visual quality
- No breaking changes to existing functionality
- Enables cinematic visual effects and atmosphere
